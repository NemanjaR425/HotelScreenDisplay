# Hotel Lobby Digital Signage Design Guidelines

## Design Approach
**Selected Approach:** Professional Hospitality Design System  
**Justification:** Digital signage requires maximum readability, professional appearance, and timeless design that reflects hotel brand standards. Drawing inspiration from premium hotel chains' digital experiences while prioritizing information clarity.

**Key Design Principles:**
- Exceptional readability from 6-15 feet viewing distance
- Welcoming yet professional hospitality aesthetic
- Seamless content transitions without distraction
- Timeless design that won't feel outdated

## Core Design Elements

### Color Palette
**Primary Colors (Dark & Light Mode):**
- Primary: 220 15% 25% (Deep navy for trust and professionalism)
- Background: 0 0% 98% (Warm white for light mode) / 220 15% 12% (Dark charcoal for dark mode)
- Text Primary: 220 15% 15% / 0 0% 95%
- Accent: 200 25% 55% (Sophisticated blue for highlights)

**Supporting Colors:**
- Success: 145 65% 45% (For positive information like weather)
- Subtle borders: 220 15% 85% / 220 15% 25%

### Typography
**Font Families:** Inter (primary), system fallbacks
**Hierarchy:**
- Hero headings: 4xl-6xl, font-weight 600
- Section headers: 2xl-3xl, font-weight 500  
- Body content: lg-xl, font-weight 400
- Time/date: 3xl-4xl, font-weight 300

### Layout System
**Spacing Units:** Tailwind units of 4, 6, 8, 12, 16 (consistent grid-based spacing)
**Grid Structure:** Full-screen sections with generous padding, content centered within safe viewing zones

### Component Library

**Core Components:**
- **Welcome Header:** Large hotel name/logo with current date/time
- **Content Cards:** Rounded, subtle shadows, generous padding (p-8, p-12)
- **Information Panels:** Clean data display with clear labels
- **Slide Indicators:** Subtle dots showing current content position
- **Weather Widget:** Icon + temperature + conditions in compact format

**Navigation:** Auto-rotating content with 8-10 second intervals per slide

**Content Sections:**
1. Welcome/time/weather display
2. Hotel amenities showcase with icons
3. Local attractions with brief descriptions
4. Hotel services and contact information

### Visual Treatments
**Card Design:** Subtle elevation with soft shadows, rounded corners (rounded-xl)
**Icons:** Use Heroicons outline style for consistency and clarity
**Transitions:** Fade transitions between content slides (duration-700)
**Backgrounds:** Solid colors with optional subtle gradient overlays

### Animations
**Minimal Animation Strategy:**
- Smooth fade transitions between content slides only
- No attention-grabbing animations that distract from information consumption
- Gentle slide-in effects for individual content elements (optional)

## Images
**Image Requirements:**
- **No large hero images** - prioritize information readability
- **Amenity icons:** 64x64px minimum, professional icon set
- **Weather icons:** Clear, universally recognizable symbols
- **Optional hotel logo:** Professional placement in header area
- **Background textures:** Subtle, low-contrast patterns if needed

**Image Placement:**
- Small decorative elements only
- Icons to support text content
- Maintain focus on textual information

This design ensures maximum readability while maintaining the sophisticated, welcoming atmosphere appropriate for hotel environments.
